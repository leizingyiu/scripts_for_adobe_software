# 用户使用说明书 (User Guide) — Photoshop 智能边界对齐与变换脚本 AlignAndTransform.jsx

欢迎使用 **Photoshop 智能边界对齐与变换脚本**。本工具能够通过识别特殊的图层名称，自动将一个图层组内的所有元素对齐并缩放到指定的“边界框”内。它不仅支持单向对齐，还支持复杂的两端拉伸、等比填充与适应，并能完美保护您的像素画质。

---

## 1. 快速入门

### 1.1 安装与运行
本项目为一个标准扩展脚本文件（`.jsx`），无需安装，即点即用：
1.  打开 Photoshop，并打开您需要处理的文档。
2.  点击顶部菜单栏：`文件 (File) -> 脚本 (Scripts) -> 浏览... (Browse...)`。
3.  在弹出的文件浏览器中选择 `AlignAndTransform.jsx` 脚本文件，点击确定。
4.  根据弹出的界面选择统一操作选项，点击“执行扫描对齐”即可。
5.  等待脚本运行完毕，查看结果。

> **提示**：如果您经常使用，可以将该 `.jsx` 文件直接复制到 Photoshop 安装目录下的 `Presets/Scripts` 文件夹中。重启 Photoshop 后，该脚本将直接显示在 `文件 -> 脚本` 菜单列表中。

## 2. 核心机制：如何命名边界图层？

脚本的核心是通过**图层名称**来寻找对齐的“参照物”。你只需要在某个图层组内，创建一个矩形（或其他任意图层），将其命名包含 `[[对齐指令]]:变换模式` 的格式即可。

### 2.1 基础对齐指令字典
你可以在 `[[...]]` 中使用以下符号或文字（不区分大小写，中英文及符号通用）：

| 方向 | 符号 | 英文关键字 | 中文关键字 |
| :--- | :---: | :--- | :--- |
| **上** | `^` | `top` | `上` |
| **下** | `v` | `bottom` | `下` |
| **左** | `<` | `left` | `左` |
| **右** | `>` | `right` | `右` |
| **中心** | `*` | `middle` / `center` | `中` |

### 2.2 单向与中心对齐示例
* `[[<]] 参考框` 或 `[[左]] rect`：组内其他图层将全部向该图层的**左边缘**对齐。
* `[[^]] rect`：组内其他图层全部向该图层的**上边缘**对齐。
* `[[*]] rect`：组内其他图层全部移动到该图层的**正中心（水平且垂直居中）**。
* `[[<^]] rect`：组内其他图层全部对齐到该图层的**左上角**。

### 2.3 两端对齐与缩放变换（核心功能）
当你在括号内**同时输入相反方向**的指令时（例如“左”和“右”，或者“上”和“下”），脚本会判定对齐方式为“两端对齐”。此时，组内其他图层不仅会移动，还会根据你指定的模式进行**缩放变换**：

通过在括号后加上冒号 `:`，你可以为这个组单独指定三种变换操作之一：

#### ① 拉伸变形 (`:stretch` 或 `:拉伸`)
* **命名示例**：`[[左右]]:stretch` 或 `[[<>v^]]:拉伸`
* **效果**：强制目标图层的边缘贴紧边界框。如果边界框的宽高比与图层不一致，图层会被**非等比拉伸变形**。

#### ② 等比适应 (`:fit` 或 `:适应`)
* **命名示例**：`[[左右上下]]:fit` 或 `[[<>v^]]:适应`
* **效果**：图层会保持原本的宽高比例进行缩放，直到某一侧长边或宽边触碰到边界框边缘。图层内容会**完整展现**在边界框内，绝对不会被裁剪。

#### ③ 等比填充 (`:fill` 或 `:填充`)
* **命名示例**：`[[左右]]:fill` 或 `[[<>v^]]:填充`
* **效果**：图层会保持原本的宽高比例进行缩放，直到整个边界框被**完全填满**。如果图层比例与边界框不一致，图层超出边界的部分会被视作裁剪溢出（常用于头像、全景背景图占位）。

## 3. 高级图层结构与匹配规则

### 3.1 自底向上嵌套结构
本脚本采用“自底向上（Inside-Out）”的规则。如果你有多个组相互嵌套，脚本会**先处理最内层的组，然后再处理外层的组**。

**正确结构示例**：
```text
▼ 组-大背景 [[左右]]:fit  <-- (3) 最后处理最外层，将“组-内层元素”整体作为一个对象在外部大框内做等比适应
  ├── 矢量大背景框
  ▼ 组-内层元素          <-- (2) 其次处理该层，使其内部的文本和图片对齐矩形2
    ├── 文本图层A
    ├── [[*]] 矩形2      <-- (1) 最先找到并激活内层的正中心对齐
  └── [[左右]]:fill 矩形1
```

### 3.2 运行时的全局弹窗配置
当脚本扫描到文档中含有需要“两端对齐”的普通像素图层（Bitmap）或尺寸不够放大的智能对象时，会弹出配置面板：

* **默认两端对齐变换模式**：如果你的图层名只写了 `[[左右]]` 而没有加后缀，在这里选择点击的模式（拉伸/适应/填充）将作为全局默认操作。
* **像素图层(Bitmap)及尺寸不足的智能对象的处理方案**：
    * **转换为智能对象（强烈推荐）**：脚本在拉伸前会自动把你的普通像素图层转换为智能对象。这样即使后续再次拉伸或缩小，也不会破坏原始图层的分辨率。
    * **不转换，直接允许拉伸**：不进行转换，直接对其像素执行强行拉伸（可能会导致像素模糊）。
    * **跳过变换操作**：这些像素图层或尺寸不够的智能对象将只执行单向对齐或居中位移，**不进行任何大小缩放**。

## 4. 常见问题与注意事项

* **问：为什么提示对齐完成，但是图层没有发生缩放？**
    * 答：请检查你的对齐指令是否构成了“两端”。必须同时包含相反方向（如 `左+右` 或 `上+下` 或 `^和v`）才会触发缩放。如果只有单方向（如 `[[左]]`），则只会发生位移，不会发生缩放。
* **问：如何一键撤销脚本做的所有更改？**
    * 答：脚本已集成历史记录保护，运行后只需在 Photoshop 内按下 `Ctrl + Z` (Windows) 或 `Cmd + Z` (Mac)，即可一次性回滚到运行脚本前的状态。
* **问：图层组里可以有多个 `[[...]]` 边界图层吗？**
    * 答：同一个图层组层级内**只能有一个**边界图层。如果存在多个，脚本只会识别排在最上方的第一个边界层。如有复杂的多重对齐需求，请通过分多层建立“图层组”嵌套来实现。
